# Aplikasi Pengelolaan dan Maintenance Alat Multimedia HM Official

Aplikasi web sederhana berbasis PHP + MySQL untuk mengelola inventaris alat multimedia, maintenance, permohonan pembelian, pengeluaran, dan konten multimedia.

Persyaratan:

- PHP 7.4+ (disarankan PHP 8)
- MySQL / MariaDB
- XAMPP / Laragon atau web server lokal

Instalasi singkat:

1. Salin folder `multimedia-hm` ke `C:\laragon\www` atau `C:\xampp\htdocs`.
2. Buat database dengan menjalankan file `install.sql` (gunakan phpMyAdmin atau mysql client).
3. Sesuaikan kredensial database di `config.php` jika perlu.
4. Akses aplikasi via http://localhost/multimedia-hm

Akun contoh:

- admin / admin123
- staff / staff123

Catatan eksport PDF: tombol "Export PDF" membuka halaman yang dioptimalkan untuk dicetak; gunakan fitur Print > Save as PDF di browser.

Jika ingin menambahkan PDF server-side, bisa memasang library seperti FPDF / TCPDF dan menyesuaikan endpoint ekspor.
